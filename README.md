Sismo data source is an off-chain infrastructure which allow to generate Attesters available groups used in the Sismo protocol.

## Lists

Lists of accounts are reusable tools used to generate Groups. 

These groups could be generated by filtering lists, creating unions, intersections and more complex operations.

Anyone can propose a new list to Sismo. 

- Feel free to open a PR 
- Join us on [discord](https://discord.gg/sismo) 

**Generate a list in local**
``` sh
yarn generate:list sismo-POAPs
```

## Attester groups

In order to allow users to build attestations from a list we need to generate a group from this list.

This part of the data source is not open source yet.

Here is a code snippet of what it will look like :

``` javascript
export default new AttesterGroupsGenerator({
  name: "merkle-public-attester",
  generate: async () => {
    const list = await ListDocument.getLatestList({generatorName: "ens-voters"})
    const accountsTree = accountsMerkleTree(list);
    const group = new Group({
      id: accountsTree.getRoot(),
      data: accountsTree, 
    });
    return [group];
  },
});
```

If you want to generate a group please send us a message on [discord](https://discord.gg/sismo).
